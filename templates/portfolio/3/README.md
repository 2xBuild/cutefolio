# Portfolio 3 — Sleek

Self-contained template inspired by [ramxcodes/sleek-portfolio](https://github.com/ramxcodes/sleek-portfolio).

## Template-specific assets (no main app pollution)

- **sleek.css** — All styles (Hanken Grotesk from CDN, 3D shadows, View Transitions)
- **theme-toggle.tsx** — Theme toggle with View Transitions API
- **navbar.tsx** — Sticky navbar with section links
- **oneko-cat.tsx** — Cat following cursor ([oneko.js](https://github.com/adryd325/oneko.js))
- **oneko/** — oneko.ts + oneko.gif (loaded only when this template is active)

## Dependencies

- None. Font from CDN, oneko from bundled script. Uses shared `@/components/ui`, `@/lib/icons`, etc.
